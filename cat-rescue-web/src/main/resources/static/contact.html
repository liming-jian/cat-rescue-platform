常见问题

领养人需年满18周岁，有稳定住所和经济来源。需签署领养协议，承诺科学喂养、定期体检和绝育。领养免费，需缴纳200元押金（完成绝育后退还）。

进入"志愿者"页面填写报名表。每月第一个周末举办新志愿者培训，内容包括猫咪基础护理、TNR操作规范和基地日常工作流程。

请拨打24小时救助热线 021-1234-5678，或通过微信公众号发送定位和照片。紧急情况请先就近送往合作宠物医院，费用由平台承担。

所有义卖收入100%用于流浪猫救助：医疗费用约60%，日常喂养约25%，基地运营约15%。每月在公众号公示收支明细。
