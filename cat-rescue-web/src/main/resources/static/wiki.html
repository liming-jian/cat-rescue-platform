TNR 绝育

什么是 TNR 计划？

TNR 是 Trap（诱捕）、Neuter（绝育）、Return（放归）的缩写。这是目前国际公认的控制流浪猫数量最人道、最有效的方法。

科学喂养

秋冬季节流浪猫投喂指南

冬天流浪猫急需高热量的食物。请避免喂食含盐量高的人类食物，尽量提供干粮和清洁的温水。注意为猫咪提供避风保暖的庇护所，防止冻伤耳尖和脚垫；增加室内互动游戏时间，帮助它们消耗精力、保持活力。

医疗急救

遇到生病的流浪猫怎么办？

如果发现流浪猫眼部分泌物多、流鼻涕或行动不便，这可能是猫鼻支或受伤。请不要盲目用药，先尝试联系当地救助组织。

领养指南

领养一只流浪猫，你需要准备什么？

除了猫砂盆、猫粮等物资，更重要的是做好"不离不弃"的心理准备，以及封窗（防止猫咪坠楼）的安全措施。
