暖心救助日记

每一次相遇都是奇迹，记录从流浪到回家的全过程。

2023年11月15日

雨夜中的"暖暖"

那是一个暴雨的夜晚，志愿者在小区的车底发现了一只全身湿透、瑟瑟发抖的小橘猫。它甚至没有力气发出声音。我们立即将它送往医院，经过三天的保温和输液，它终于吃下了第一口罐头。现在它叫"暖暖"，是个黏人的小胖子。

2023年9月02日

废墟中的三花猫妈妈

在即将拆迁的工地废墟里，一只三花母猫死死守着它的一窝小猫不肯离去。面对轰鸣的挖掘机，它显得那么无助。我们的救援队连夜行动，使用了诱捕笼，成功在拆迁前一小时将一家五口全部救出。现在小猫们都已经找到了领养家庭。

2023年6月20日

从哈气到露肚皮：警长"黑猫"的转变

"黑猫"是社区里出了名的凶猫，谁靠近就挠谁。其实它是因为曾经被人伤害过，才会如此应激。志愿者花了整整三个月，每天定点投喂，慢慢坐得离它近一点。直到那一天，它主动用头蹭了蹭志愿者的手，那一刻，所有的付出都值了。

2023年3月10日

只有三条腿的勇士

因为车祸，小狸花永远失去了它的右后腿。但截肢手术后的第二天，它就坚强地站了起来，努力适应新的行走方式。它的乐观感染了所有人，现在它被一对退休夫妇收养，是家里的开心果。
